# 🚀 AWS Deployment Demo – Next.js App (EC2)

This project demonstrates my ability to deploy a Next.js application on an AWS EC2 server and make it live using a public IP.

## 🔧 What I did:
- Launched and configured an EC2 (Ubuntu) instance
  
- Deployed a Next.js app manually via SSH
  
- Configured inbound rules to allow access
  
- Used PM2 + Nginx to run the app in production
  
- Made the app accessible via public IPv4
